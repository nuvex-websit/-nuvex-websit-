# Nuvex - نوفيكس

## موقع إنشاء المتاجر الإلكترونية وتصميم المواقع

موقع احترافي لشركة نوفيكس المتخصصة في إنشاء المتاجر الإلكترونية وتصميم المواقع الاحترافية.

### 🌟 المميزات

- ✅ تصميم متجاوب وحديث
- ✅ دعم اللغة العربية كاملاً (RTL)
- ✅ موقع محسّن لمحركات البحث (SEO)
- ✅ أداء سريع وخفيف
- ✅ سهل التخصيص والتطوير
- ✅ نموذج توصل متكامل
- ✅ أيقونات من Font Awesome
- ✅ رابط واتساب مباشر

### 📁 بنية الملفات

```
nuvex-website/
├── index.html          # الصفحة الرئيسية
├── styles.css          # أنماط الموقع
├── script.js           # سكريبتات الموقع
├── logo.png           # شعار الشركة
└── README.md          # هذا الملف
```

### 📋 الأقسام الرئيسية

1. **الرأس (Header)** - تنقل سهل بين الأقسام المختلفة
2. **البطل (Hero)** - صورة ترحيبية جذابة
3. **من نحن (About)** - معلومات عن الشركة والإحصائيات
4. **الخدمات (Services)** - عرض الخدمات المختلفة
5. **أعمالنا (Portfolio)** - معرض أعمال سابقة
6. **آراء العملاء (Testimonials)** - تقييمات العملاء
7. **الباقات (Pricing)** - خطط الأسعار المختلفة
8. **التواصل (Contact)** - نموذج اتصال مباشر
9. **التذييل (Footer)** - روابط سريعة ومعلومات

### 🎨 الألوان المستخدمة

- **Navy**: #1A2B4C (الأزرق الداكن)
- **Teal**: #00A5A5 (الأزرق المائي)
- **Light**: #F8FAFC (الأبيض الفاتح)
- **Gray**: #64748B (الرمادي)

### 🚀 كيفية الاستخدام

#### التثبيت المحلي

```bash
# استنساخ المستودع
git clone https://github.com/nuvex-websit/-nuvex-websit-.git

# الدخول للمجلد
cd -nuvex-websit-

# فتح الملف في المتصفح
open index.html
```

#### التخصيص

1. **تغيير الشعار**: استبدل `logo.png` بشعارك
2. **تحديث المعلومات**: عدّل البيانات في `index.html`
3. **تعديل الألوان**: غيّر متغيرات CSS في `styles.css`
4. **إضافة صور**: استبدل صور المجموعات في قسم Portfolio
